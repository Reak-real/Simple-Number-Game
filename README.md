# Zahlen-Rater 🎲

Ein einfaches Zahlen-Ratespiel in C++.

## Was macht es?

Der Computer denkt sich eine zufällige Zahl zwischen 1 und 100 aus. Du musst sie erraten – er gibt dir nach jedem Versuch einen Tipp.

## Starten

1. Projekt in CLion öffnen
2. Play-Button ▶️ drücken
3. Zahl eingeben und Enter drücken

## Beispiel

```
Rate eine Zahl zwischen 1 und 100: 50
Zu niedrig!
Rate eine Zahl zwischen 1 und 100: 75
Zu hoch!
Rate eine Zahl zwischen 1 und 100: 63
Gewonnen!
```

## Gebaut mit

- C++20
- CLion
